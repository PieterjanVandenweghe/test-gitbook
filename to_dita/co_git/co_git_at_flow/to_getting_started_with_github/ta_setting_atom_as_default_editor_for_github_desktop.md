# Setting Atom as default editor for GitHub Desktop

1. In GitHub Desktop, choose **File** &gt; **Options**.
2. Choose the **Integrations** tab and select **Atom** as **External editor**.

   ![](../../../../.gitbook/assets/git-options-integrations.png)

3. Click **Save**.
