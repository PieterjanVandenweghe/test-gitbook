--- authorinformation: --- # Staging changes This task explains how to stage specific changes. **Note:** You can also skip the staging area. 1. Type one of the following: - `git add `: Stage changes made to one specific file. - `git add -A`: Stage all changes, including new and deleted files. - `git add .`: Stage new files and modifications, without deleted files. - `git add -u`: Stage modifications and deletions, without new files. 2. Press Enter.
