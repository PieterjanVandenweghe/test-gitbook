--- authorinformation: --- # Flow bv Schematron checks on GitHub The Schematron checks are stored in the **DITA** repository on GitHub. You can find them in the Schematron/rules folder. If you are not a member of the Flow bv GitHub team, contact the administrator to get the files. Once downloaded, save the folder rules in the \_custom\_framework/dita\_extension folder of your project repository. **Related information**
