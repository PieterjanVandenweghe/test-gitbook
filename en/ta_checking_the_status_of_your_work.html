--- authorinformation: [, Yves.Barbion] --- # Checking the status of your work To check the changes you have made since the last commit on your branch, do the following: 1. Type `git status`. 2. Press Enter. The status of the current active branch is shown. ![](../_media/graphics/status_staged_changes.png) In the example above, new files have been created. These additions have already been [staged](ta_staging_changes.html) and are ready to be [committed](ta_committing_changes.html). One other newly created file has not been staged yet. ![](../_media/graphics/status.png) In the example above, all changes have been committed, but there are still a number of commits to be [pushed](ta_pushing_a_commit.html) to the remote repository.
