This script referenced to a file service-worker.js for testing purposing. After installing and configuring the Workbox configuration, the service-workers.js became a template file for the sw.js in which all files to be cached are stored. This means we also have to update our script in the index.html file. 1. Open the index.html page in a text editor. 2. Search for `navigator.serviceWorker.register('/service-worker.js');` and replace it with `navigator.serviceWorker.register('/sw.js');`. 3.
